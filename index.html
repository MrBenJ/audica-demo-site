Audica

Extract your audio frequency/bytetime data to create amazing visualizations

Example

            
              const myAudica = audica({
                element: document.getElementById('my-audio-or-video-element')
                size: 256,
                dataType: 'time' // or 'hz' for frequency
              });

              const loop = () => {
                window.requestAnimationFrame(loop);

                const data = myAudica.getData();

                for (let i = 0; i < 256; i++) {
                  // data is an array (length 256) of integers
                  // it contains byte time data to show
                  // an audio wave.

                  // Setting "dataType" to "hz" returns frequency data instead.
                  // This allows you to measure Bass/Mid/Treble values in the array,
                  // with the first values being more bass, and last values being treble

                }
              }

Install via unpkg

          
            <script src="https://unpkg.com/audica"></script>
